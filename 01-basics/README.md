## Setup Environment on MacOS
```
$ brew install node
$ node -v
v32.10.0
$ npm -v
10.9.2
```

Run JavaScript by Node
```shell
$ node practice.js
```

```shell
$ npm install -g typescript
$ tsc -v
```

[Optional]
```shell
# Make tsconfig.json
$ tsc --init
```

Run TypeScript
```shell
$ tsc && node dist/app.js
```
